# 설유화(sullyuwha) — 실크 기성복 디자인 시스템

> **설유화**는 실크를 일상으로 들이는 기성복 브랜드다.
> 디자인은 **모던 럭셔리 미니멀** — 실크의 광택과 드레이프, 고요한 여백, 절제된 골드 포인트.
> 장식을 덜어내고 소재와 실루엣이 스스로 말하게 한다.

---

## 1. 디자인 철학

실크는 그 자체로 완성된 소재다. 페이지도 그래야 한다 — **덜어내고, 비우고, 흐르게.**

| 원리 | 웹 디자인 번역 |
| --- | --- |
| 실크의 광택 | 부드러운 명암 대비, 큰 이미지, 은은한 골드 하이라이트 |
| 드레이프(늘어짐) | 느린 이징, 세로로 흐르는 레이아웃, 여유로운 행간 |
| 여백의 고요 | 관대한 섹션 간격, 중앙 정렬 에디토리얼 구성 |
| 절제 | 색·장식 최소화. 화면의 대부분은 아이보리/차콜 |
| 촉감 | 종이 질감 대신 매끈한 표면, 미세한 그라디언트 |

**핵심 원칙**
1. **소재 우선** — 이미지가 주인공. UI는 액자 역할에 머문다.
2. **여백** — 요소 사이를 넉넉히. 비움이 곧 고급감.
3. **절제된 강조** — 골드는 포인트로만(하이라이트·라인·소량 텍스트).
4. **에디토리얼** — 얇은 세리프 제목 + 넓은 자간의 라틴 라벨로 매거진 톤.

---

## 2. 색상 팔레트

무채색 뉴트럴 베이스에 골드 하나. 채도는 낮게, 톤은 따뜻하게.

### 기본(Neutral)

| 토큰 | HEX | 이름 | 용도 |
| --- | --- | --- | --- |
| `--color-ivory` | `#F7F3EC` | 아이보리 | 기본 배경 |
| `--color-champagne` | `#E7DFD0` | 샴페인 | 카드/교차 섹션 배경 |
| `--color-mist` | `#EFE9DE` | 미스트 | 아주 옅은 톤 구분 |
| `--color-charcoal` | `#2A2724` | 차콜 | 기본 텍스트, 다크 섹션 배경 |
| `--color-taupe` | `#7C7367` | 토프 | 보조 텍스트 |
| `--color-line` | `#E0D8C8` | 라인 | 헤어라인 구분선/보더 |

### 강조(Accent)

| 토큰 | HEX | 이름 | 용도 |
| --- | --- | --- | --- |
| `--color-gold` | `#B79363` | 골드 | 포인트, 라벨, 하이라이트, 호버 |

> **사용 비율**: 아이보리/차콜 계열 **90%**, 골드 **10% 이하**.
> 골드는 "시선을 한 번 멈추게 할 때"만. 면적으로 쓰지 않는다.

### 다크 섹션 (Hero / 소재 이야기 / Footer)

차콜 배경 위에 아이보리 텍스트, 골드 포인트. 실크 사진의 명암이 살아난다.

---

## 3. 타이포그래피

에디토리얼 럭셔리의 핵심. **얇은 세리프 제목** + **넓은 자간의 라틴 라벨** + **가벼운 산세리프 본문**.

| 역할 | 서체 | 굵기 | 비고 |
| --- | --- | --- | --- |
| Display / 제목 | `Cormorant Garamond` → `Noto Serif KR` | 300·400 | 라틴은 코모란트, 한글은 노토 세리프로 자동 폴백 |
| Eyebrow / 라벨 | `Noto Sans KR` (uppercase) | 400 | 대문자 + `tracking` 넓게 (예: COLLECTION) |
| 본문 / UI | `Noto Sans KR` | 300·400·500 | 가볍고 정갈하게 |

### 타입 스케일 (모바일 → 데스크톱, `clamp`)

| 토큰 | 크기 | 용도 |
| --- | --- | --- |
| Display | `clamp(2.75rem, 7vw, 5.5rem)` | Hero 헤드라인 (아주 크게) |
| H2 | `clamp(1.9rem, 4vw, 3.25rem)` | 섹션 제목 |
| H3 | `clamp(1.15rem, 2vw, 1.5rem)` | 카드 제목 |
| Body | `clamp(0.95rem, 1.05vw, 1.0625rem)` | 본문 |
| Label | `0.75rem` (uppercase, `letter-spacing: 0.2em`) | Eyebrow |

### 조판 규칙 (필수)

```css
word-break: keep-all;        /* 한글 어절 단위 줄바꿈 */
overflow-wrap: break-word;
```

- 제목 `line-height`: **1.15–1.25**, 본문: **1.8**.
- 세리프 제목 자간: `-0.01em`. 라틴 라벨: `+0.2em`.
- 본문 최대 폭: **`60ch`**. 얇은 세리프는 굵기 300~400로 가늘게 유지.

---

## 4. 여백 · 레이아웃

럭셔리는 여백에서 나온다. 8px 그리드, 섹션 간격은 크게.

| 토큰 | 값 |
| --- | --- |
| 콘텐츠 최대폭 | `1240px` (`--container`) |
| 좁은 텍스트 폭 | `720px` (에디토리얼 문단/헤딩) |
| 좌우 거터 | 모바일 `20px` / 데스크톱 `48px` |
| 섹션 상하 패딩 | `clamp(5rem, 12vw, 10rem)` |
| 요소 간 리듬 | `8 / 16 / 24 / 40 / 72 / 120 px` |

- 섹션 제목은 **중앙 정렬** 기본(에디토리얼). 상품/정보는 그리드 좌측 정렬.
- 이미지는 여백을 두고 크게. 경계는 얇은 라인 또는 무테.
- 모서리 `border-radius`는 **0~2px**(각지고 정갈하게).

---

## 5. 모티프 · 디테일

미니멀. 장식은 거의 없고, 있는 것은 얇고 정밀하다.

- **골드 헤어라인** — eyebrow 아래/구분선에 1px 골드 짧은 선.
- **넓은 자간 라틴 라벨** — COLLECTION · SILK · LOOKBOOK · CONTACT.
- **대형 실크 이미지** — 드레이프·광택이 보이도록 크게, 여백과 함께.
- **호버** — 이미지 살짝 확대(scale 1.03), 골드 언더라인 등장. 느리게.
- **플레이스홀더** — 샴페인→아이보리 부드러운 그라디언트 + 아주 옅은 모노그램(설). 과하지 않게.

---

## 6. 컴포넌트 스타일 규칙

### 버튼
- **Primary**: 차콜 배경 / 아이보리 글자, 호버 시 골드 배경.
- **Ghost(선)**: 투명 / 차콜 얇은 보더(1px) / 차콜 글자, 호버 시 차콜 채움.
- 패딩 `15px 34px`, `border-radius: 2px`, **uppercase + `letter-spacing: 0.12em`**.
- 트랜지션 `320ms` 느린 이징.

### 상품 카드 (컬렉션)
- 무테 또는 얇은 라인. 이미지 상단(세로 3:4), 아래 카테고리 라벨 · 이름 · 소재 · 짧은 설명.
- 호버: 이미지 scale 1.03, 이름에 골드 언더라인.

### 섹션 헤딩
- 위: 골드 짧은 선 + 넓은 자간 라틴 eyebrow.
- 아래: 얇은 세리프 큰 제목(중앙 정렬 기본). 필요 시 한 줄 서브카피.

### 이미지 (R2)
- 모든 이미지는 Cloudflare R2 공개 버킷 + 커스텀 도메인 서빙, `next/image` 최적화.
- 미설정/로딩 시 샴페인 그라디언트 플레이스홀더로 대체.

---

## 7. 모션

실크처럼 느리고 매끄럽게.

- 기본 이징: `cubic-bezier(0.22, 1, 0.36, 1)`.
- 진입: `opacity` + `translateY(24px)` → 0, `800ms`, 한 번만(`once`).
- 이미지 호버 확대 `600ms`. 과한 패럴랙스·바운스 금지.
- `prefers-reduced-motion` 존중 — 모션 최소화 시 즉시 표시.

---

## 8. 반응형 브레이크포인트

| 이름 | 폭 | 대상 |
| --- | --- | --- |
| base | `< 640px` | 모바일 (1컬럼) |
| `sm` | `≥ 640px` | 큰 모바일 |
| `md` | `≥ 768px` | 태블릿 (2컬럼) |
| `lg` | `≥ 1024px` | 데스크톱 (3컬럼) |
| `xl` | `≥ 1280px` | 와이드 |

- **모바일 퍼스트**. 데스크톱에서 여백·타입 스케일 확장.
- 터치 타깃 최소 `44px`. 헤더는 모바일에서 미니멀 메뉴(햄버거 → 오버레이).
- 이미지는 `sizes`로 뷰포트별 해상도 최적화.

## 9. 페이지 구조 (섹션 순서)

실크 기성복의 이야기를 위→아래로.

1. **Hero** — 브랜드 슬로건 + 대표 룩(대형 실크 이미지, 차콜 배경).
2. **브랜드 소개** — 설유화의 지향점, 이름의 의미.
3. **컬렉션** — 실크 블라우스·슬립 원피스·셋업·스카프 등 라인업 그리드.
4. **실크 소재 이야기** — 광택/통기성/원사 등급/홈 케어 (다크 배경, 피처 카드).
5. **룩북 갤러리** — 착장·디테일 컷 메이슨리 그리드.
6. **문의 · 스토어** — 쇼룸/구매·입점 문의, 연락처, 위치 (다크 배경).
7. **푸터** — 워드마크, SNS, 저작권.

---

## 10. 접근성 · 품질 기준

- 명도 대비 **WCAG AA** 이상 (본문 4.5:1, 큰 글자 3:1). 토프 텍스트는 배경 대비 확인.
- 모든 이미지 `alt` 필수, 장식 이미지는 `alt=""`.
- 키보드 포커스 링 표시(골드).
- 시맨틱 HTML(`<header> <main> <section> <footer>`), 섹션마다 `aria-labelledby`.
- `word-break: keep-all` 전역 적용으로 한글 어절 단위 줄바꿈 보장.
