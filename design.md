# 설유화(sullyuwha) — 한복 예복 디자인 시스템

> **설유화**는 전통의 아름다움 위에 현대의 감각을 더해 단 하나의 예복을 짓는 한복 예복·맞춤 브랜드다.
> 디자인은 **크림 톤 미니멀** — 부드러운 오트밀 배경, 여백의 미, 설유화 꽃가지의 흰 부조(엠보스), 절제된 브론즈 포인트.
> 장식을 덜어내고 선과 소재가 스스로 말하게 한다.

---

## 1. 디자인 철학

예복은 그 자체로 완성된 격을 지닌다. 페이지도 그래야 한다 — **덜어내고, 비우고, 흐르게.**

| 원리 | 웹 디자인 번역 |
| --- | --- |
| 전통의 기품 | 얇은 세리프 제목, 크림 톤, 절제된 브론즈 |
| 예복의 선 | 여유로운 행간, 세로로 흐르는 레이아웃 |
| 여백의 미 | 관대한 섹션 간격, 설유화 꽃가지가 여백에 스밈 |
| 절제 | 색·장식 최소화. 화면의 대부분은 크림/오트밀 |
| 부조의 질감 | 흰 꽃가지를 이중 그림자로 도드라지게(엠보스) |

**핵심 원칙**
1. **소재·선 우선** — 이미지와 예복의 선이 주인공. UI는 액자 역할에 머문다.
2. **여백** — 요소 사이를 넉넉히. 비움이 곧 고급감.
3. **절제된 강조** — 브론즈는 포인트로만(하이라이트·라인·소량 텍스트·아이콘).
4. **에디토리얼** — 얇은 세리프 제목 + 넓은 자간의 라틴 라벨로 매거진 톤.

---

## 2. 색상 팔레트

따뜻한 크림/오트밀 뉴트럴 베이스에 브론즈 하나. 채도는 낮게, 톤은 따뜻하게. 전 페이지가 밝다(다크 섹션 없음).

### 기본(Neutral)

| 토큰 | HEX | 이름 | 용도 |
| --- | --- | --- | --- |
| `--color-ivory` | `#F3ECE0` | 아이보리(크림) | 기본 배경 |
| `--color-champagne` | `#E7DCC8` | 샴페인 | 카드/교차 섹션 배경 |
| `--color-mist` | `#EEE6D8` | 미스트 | 아주 옅은 톤 구분 |
| `--color-sand` | `#DED0B8` | 샌드 | 카테고리 카드/살짝 진한 베이지 |
| `--color-charcoal` | `#3D352B` | 에스프레소 | 기본 텍스트(따뜻한 브라운) |
| `--color-taupe` | `#8A7C69` | 토프 | 보조 텍스트 |
| `--color-line` | `#E1D7C3` | 라인 | 헤어라인 구분선/보더 |

### 강조(Accent)

| 토큰 | HEX | 이름 | 용도 |
| --- | --- | --- | --- |
| `--color-gold` | `#AB8F67` | 브론즈 | 포인트, 라벨, 아이콘, 하이라이트 |

> **사용 비율**: 크림/오트밀 계열 **90% 이상**, 브론즈 **10% 이하**.
> 브론즈는 "시선을 한 번 멈추게 할 때"만. 면적으로 쓰지 않는다.

### 히어로 — 흰 부조(엠보스) 꽃가지

크림 배경 위에 **설유화 꽃가지**(`public/branch-relief.png`, 흰 실루엣)를 얹고,
`filter`의 이중 `drop-shadow`(어두운 그림자 + 밝은 하이라이트)로 부조처럼 도드라지게 한다.
어른거리는 나뭇잎 그림자를 아주 옅게 겹쳐 햇살의 결을 더한다. 여백의 미를 살린 첫인상.

## 3. 타이포그래피

에디토리얼 럭셔리의 핵심. **얇은 세리프 제목** + **넓은 자간의 라틴 라벨** + **가벼운 산세리프 본문**.

| 역할 | 서체 | 굵기 | 비고 |
| --- | --- | --- | --- |
| Display / 제목 | `Cormorant Garamond` → `Noto Serif KR` | 300·400 | 라틴은 코모란트, 한글은 노토 세리프로 자동 폴백 |
| Eyebrow / 라벨 | `Noto Sans KR` (uppercase) | 400 | 대문자 + `tracking` 넓게 (예: COLLECTION) |
| 본문 / UI | `Noto Sans KR` | 300·400·500 | 가볍고 정갈하게 |

### 타입 스케일 (모바일 → 데스크톱, `clamp`)

| 토큰 | 크기 | 용도 |
| --- | --- | --- |
| Display | `clamp(2.75rem, 7vw, 5.5rem)` | Hero 헤드라인 (아주 크게) |
| H2 | `clamp(1.9rem, 4vw, 3.25rem)` | 섹션 제목 |
| H3 | `clamp(1.15rem, 2vw, 1.5rem)` | 카드 제목 |
| Body | `clamp(0.95rem, 1.05vw, 1.0625rem)` | 본문 |
| Label | `0.75rem` (uppercase, `letter-spacing: 0.2em`) | Eyebrow |

### 조판 규칙 (필수)

```css
word-break: keep-all;        /* 한글 어절 단위 줄바꿈 */
overflow-wrap: break-word;
```

- 제목 `line-height`: **1.15–1.25**, 본문: **1.8**.
- 세리프 제목 자간: `-0.01em`. 라틴 라벨: `+0.2em`.
- 본문 최대 폭: **`60ch`**. 얇은 세리프는 굵기 300~400로 가늘게 유지.

## 4. 여백 · 레이아웃

럭셔리는 여백에서 나온다. 8px 그리드, 섹션 간격은 크게.

| 토큰 | 값 |
| --- | --- |
| 콘텐츠 최대폭 | `1240px` (`--container`) |
| 좁은 텍스트 폭 | `720px` (에디토리얼 문단/헤딩) |
| 좌우 거터 | 모바일 `20px` / 데스크톱 `48px` |
| 섹션 상하 패딩 | `clamp(5rem, 12vw, 10rem)` |
| 요소 간 리듬 | `8 / 16 / 24 / 40 / 72 / 120 px` |

- 섹션 제목은 **중앙 정렬** 기본(에디토리얼). 상품/정보는 그리드 좌측 정렬.
- 이미지는 여백을 두고 크게. 경계는 얇은 라인 또는 무테.
- 모서리 `border-radius`는 **0~2px**(각지고 정갈하게).

## 5. 모티프 · 디테일

미니멀. 장식은 거의 없고, 있는 것은 얇고 정밀하다.

- **브론즈 헤어라인** — eyebrow 옆/구분선에 1px 브론즈 짧은 선.
- **넓은 자간 라틴 라벨** — BRAND · COLLECTION · BESPOKE · STORY · CONTACT.
- **라인 엠블럼 아이콘** — 매화 엠블럼(로고) + 특징 4종(TRADITION/CRAFTSMANSHIP/BESPOKE/HERITAGE)
  + 제작 과정 5종. `Icons.tsx`의 `currentColor` 스트로크 인라인 SVG.
- **흰 부조 꽃가지** — 히어로 배경. 흰 실루엣 + 이중 그림자로 부조처럼 도드라진다(§2 참고).
- **명함(네임카드)** — 문의 섹션에 반전한 설유화 꽃가지 컬러 사진(흰 배경은 `mix-blend-multiply`로 카드에 녹임)과
  좌측 상단 세로쓰기(`writing-mode: vertical-rl`) 타이포 — 설유화 / 고귀함이 · 기품있는 선.
- **호버** — 이미지 살짝 확대(scale 1.03~1.04), 링크에 브라운 언더라인 + 화살표 이동. 느리게.
- **플레이스홀더** — 샴페인→아이보리 부드러운 그라디언트 + 아주 옅은 모노그램(설). 과하지 않게.

## 6. 컴포넌트 스타일 규칙

### 버튼
- **Primary**: 차콜 배경 / 아이보리 글자, 호버 시 골드 배경.
- **Ghost(선)**: 투명 / 차콜 얇은 보더(1px) / 차콜 글자, 호버 시 차콜 채움.
- 패딩 `15px 34px`, `border-radius: 2px`, **uppercase + `letter-spacing: 0.12em`**.
- 트랜지션 `320ms` 느린 이징.

### 상품 카드 (컬렉션)
- 무테 또는 얇은 라인. 이미지 상단(세로 3:4), 아래 카테고리 라벨 · 이름 · 소재 · 짧은 설명.
- 호버: 이미지 scale 1.03, 이름에 골드 언더라인.

### 섹션 헤딩
- 위: 골드 짧은 선 + 넓은 자간 라틴 eyebrow.
- 아래: 얇은 세리프 큰 제목(중앙 정렬 기본). 필요 시 한 줄 서브카피.

### 이미지 (R2)
- 모든 이미지는 Cloudflare R2 공개 버킷 + 커스텀 도메인 서빙, `next/image` 최적화.
- 미설정/로딩 시 샴페인 그라디언트 플레이스홀더로 대체.

---

## 7. 모션

실크처럼 느리고 매끄럽게.

- 기본 이징: `cubic-bezier(0.22, 1, 0.36, 1)`.
- 진입: `opacity` + `translateY(24px)` → 0, `800ms`, 한 번만(`once`).
- 이미지 호버 확대 `600ms`. 과한 패럴랙스·바운스 금지.
- `prefers-reduced-motion` 존중 — 모션 최소화 시 즉시 표시.

## 8. 반응형 브레이크포인트

| 이름 | 폭 | 대상 |
| --- | --- | --- |
| base | `< 640px` | 모바일 (1컬럼) |
| `sm` | `≥ 640px` | 큰 모바일 |
| `md` | `≥ 768px` | 태블릿 (2컬럼) |
| `lg` | `≥ 1024px` | 데스크톱 (3컬럼) |
| `xl` | `≥ 1280px` | 와이드 |

- **모바일 퍼스트**. 데스크톱에서 여백·타입 스케일 확장.
- 터치 타깃 최소 `44px`. 헤더는 모바일에서 미니멀 메뉴(햄버거 → 오버레이).
- 이미지는 `sizes`로 뷰포트별 해상도 최적화.

## 9. 페이지 구조 (섹션 순서)

한복 예복의 이야기를 위→아래로. (전 페이지 밝은 크림)

1. **Hero** — 슬로건("고귀함이 서려있는 기품있는 선") + 흰 부조 설유화 꽃가지 + COLLECTION VIEW.
2. **브랜드 특징** — TRADITION · CRAFTSMANSHIP · BESPOKE · HERITAGE 4가치(라인 아이콘).
3. **About** — "시간이 지나도 변하지 않는 가치" + BRAND STORY.
4. **컬렉션** — 카테고리 그리드(WOMEN·MEN·BESPOKE·ACCESSORIES) + BEST COLLECTION(당의·스란치마 등) + 전체 보기.
5. **맞춤(Bespoke)** — "당신만을 위한 단 하나의 예복" + 자세히 보기.
6. **제작 과정(Process)** — 상담·기획 → 디자인 → 원단 → 수작업 → 피팅·완성(5단계 아이콘).
7. **문의** — 아틀리에·연락처·운영시간 + 세로 타이포 명함.
8. **푸터** — 엠블럼 워드마크, 메뉴, 저작권.

## 10. 접근성 · 품질 기준

- 명도 대비 **WCAG AA** 이상 (본문 4.5:1, 큰 글자 3:1). 토프 텍스트는 배경 대비 확인.
- 모든 이미지 `alt` 필수, 장식 이미지는 `alt=""`.
- 키보드 포커스 링 표시(골드).
- 시맨틱 HTML(`<header> <main> <section> <footer>`), 섹션마다 `aria-labelledby`.
- `word-break: keep-all` 전역 적용으로 한글 어절 단위 줄바꿈 보장.
